# <div align="center"> Hermes' Diary

  Este é o repositório para o projeto do nosso jornal (Trabalho de português), o qual denominamos "Hermes Diary" ou "Diario de Hermes, tendo seu nome baseado no deus grego Hermes, em homenagem a sua função de mensageiro do olimpo. 

{Imagem do site}

# Temas abordados

## Games e Tech
Nesse tema abordamos as novidades do mundo da tecnoligia e dos jogos eletronicos.

 {Imagem Tech e Games}

## Cultura
Nesse tema abordamos as novidades do mundo das expressões culturais, tais como musicas, filmes e séries.

 {Imagem Cultura}

## Destaques
Nesse tema abordamos os destaques gerais que ocorreram no mundo, partindo de COVID-19 até esportes.
